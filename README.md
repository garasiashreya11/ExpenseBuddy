# 🚀 ExpenseBuddy

> A Full-Stack Expense Tracking & Bill Splitting System

## 📌 About

ExpenseBuddy is a web-based expense management platform that helps users track personal expenses, categorize spending, and manage shared group expenses. The system enables bill splitting among multiple users and maintains transaction history for transparency and better financial management.

---

## ✨ Features

### 🔐 User Authentication
- User Registration
- Secure Login
- Session Management

### 💰 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Expense Categorization

### 👥 Bill Splitting
- Create Shared Expenses
- Add Group Members
- Equal Bill Distribution
- Balance Tracking

### 📊 Dashboard
- Expense Overview
- Spending Insights
- Category-wise Analysis

### 📜 Transaction History
- View Previous Transactions
- Expense Records
- Payment Tracking
