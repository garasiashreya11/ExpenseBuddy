## 📌 About The Project

ExpenseBuddy is a full-stack web application designed to help users track personal expenses, manage shared group expenses, and split bills seamlessly.

The system provides an intuitive platform for recording transactions, categorizing expenses, monitoring spending habits, and maintaining transparency among group members.

---

## ✨ Key Features

### 💰 Expense Management

* Add, edit, and delete expenses
* Categorize spending records
* Track daily and monthly expenses
* Maintain organized financial records

### 👥 Bill Splitting System

* Create shared expenses
* Add multiple group members
* Equal expense distribution
* Automatic balance calculation

### 🔐 User Authentication

* Secure user registration
* Login and logout functionality
* Session management
* Protected user accounts

### 📊 Dashboard & Analytics

* Expense overview dashboard
* Spending summaries
* Category-wise expense tracking
* Financial insights visualization

### 📜 Transaction History

* View past transactions
* Monitor payment records
* Expense activity logs
* Transparent expense management
